# 五行时尚数据标注指南

> 版本: 1.1  
> 更新日期: 2026-03-21  
> 适用范围: 衣物数据五行属性标注
> 数据格式: aligned_with_seed_data_v1.0

---

## 1. 概述

本指南用于规范衣物数据的五行属性标注流程，确保标注结果的一致性和准确性。适用于：

- 人工标注种子数据
- AI 自动标注校验
- 用户上传图片后的自动识别

## 3. 五行基础概念

### 3.1 五行元素

| 五行 | 符号 | 代表颜色 | 代表材质 | 代表风格 | 能量特征 |
|:---:|:---:|:---|:---|:---|:---|
| 金 | metal | 白、金、银 | 金属、皮革、亮片 | 简约、商务、极简 | 果断、坚毅、收敛 |
| 木 | wood | 绿、青 | 棉麻、竹纤维、天然 | 自然、森系、文艺 | 生长、仁慈、创新 |
| 水 | water | 黑、蓝 | 丝绸、雪纺、蕾丝 | 优雅、神秘、浪漫 | 智慧、灵动、深沉 |
| 火 | fire | 红、紫、橙 | 丝绒、毛呢、针织 | 个性、街头、活力 | 热情、活力、光明 |
| 土 | earth | 黄、棕、米 | 帆布、牛仔、粗织 | 复古、学院、经典 | 稳重、包容、信任 |

### 3.2 五行相生相克

```
相生: 木 → 火 → 土 → 金 → 水 → 木
相克: 木 → 土 → 水 → 火 → 金 → 木
```

## 4. 标注流程

### 4.1 标准流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  确定基础   │ -> │  查询映射   │ -> │  计算得分   │ -> │  填写数据   │
│  信息       │    │  表         │    │             │    │             │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

### 4.2 详细步骤

1. **确定基础信息**
   - 物品名称
   - 分类 (上装/下装/外套/裙装/鞋履/配饰)
   - 颜色 (名称、色值)
   - 面料 (名称、触感、克重)
   - 款式 (形状、细节)

2. **查询映射表**
   - 查询 `wuxing_color_mapping.json` 获取颜色对应五行
   - 查询 `wuxing_material_mapping.json` 获取面料对应五行
   - 查询 `wuxing_style_mapping.json` 获取款式对应五行

3. **计算五行得分**
   - 颜色贡献 = 主五行 × 能量强度 × 0.4
   - 面料贡献 = 主五行 × 0.35 + 次五行 × 0.15
   - 款式贡献 = 综合五行得分 × 0.25
   - 归一化使总和为 1.0

4. **填写数据字段**
   - 属性详情.颜色: 颜色信息
   - 属性详情.面料: 面料信息
   - 属性详情.款式: 款式信息
   - 适用标签: 场景标签

## 5. 权重计算规则

### 5.1 权重分配

| 维度 | 权重 | 说明 |
|:---|:---:|:---|
| 颜色 | 40% | 视觉冲击最强，权重最高 |
| 面料 | 35% | 主五行 35% + 次五行 15% |
| 款式 | 25% | 整体气质和搭配场景 |

### 5.2 计算公式

```
综合五行得分计算:

1. 颜色贡献 = 颜色.主五行 × 颜色.能量强度 × 0.4

2. 面料贡献:
   - 主五行贡献 = 面料.主五行 × 0.35
   - 次五行贡献 = 面料.次五行 × 0.15 (如有)

3. 款式贡献 = 款式.综合五行得分 × 0.25

4. 归一化: 使所有五行得分总和 = 1.0
```

### 5.3 计算示例

**案例: 正红色真丝衬衫 (ITEM_001)**

```json
{
  "属性详情": {
    "颜色": {"名称": "正红", "主五行": "火", "能量强度": 1.0},
    "面料": {"名称": "真丝", "主五行": "火", "次五行": "水"},
    "款式": {"综合五行得分": {"火": 0.8, "水": 0.2}}
  }
}
```

**计算过程:**
```
颜色贡献: 火 = 1.0 × 1.0 × 0.4 = 0.40
面料贡献: 火 = 0.35, 水 = 0.15
款式贡献: 火 = 0.8 × 0.25 = 0.20, 水 = 0.2 × 0.25 = 0.05

原始得分: {火: 0.95, 水: 0.20}
归一化: 总和 = 1.15
最终得分: {火: 0.83, 水: 0.17}
```

## 7. 特殊情况处理

### 6.1 多色衣物

对于多色衣物，按主次顺序处理:

1. 确定主色调 (占比 > 50%)
2. 次色调作为辅助参考
3. 如无明确主色调，取平均或按面积加权

**示例: 黑白条纹上衣**
```
主色: 黑 (水) - 50%
次色: 白 (金) - 50%
结果: wuxing_score = {water: 0.5, metal: 0.5, ...}
wuxing_element: "水" (取最高分)
```

### 6.2 混合材质

对于混合材质衣物:

1. 确定主要材质
2. 按材质占比加权计算

**示例: 棉麻混纺连衣裙**
```
棉 (木): 60%
麻 (木): 40%
结果: 木行得分 = 1.0 × 0.35 = 0.35
```

### 6.3 中性色调

对于黑、白、灰等中性色调:

| 颜色 | 默认五行 | 备注 |
|:---|:---:|:---|
| 黑色 | 水 | 深沉、神秘 |
| 白色 | 金 | 纯净、简约 |
| 灰色 | 金 | 中性，可偏向相邻五行 |
| 米色 | 土 | 温暖、稳重 |

### 6.4 争议处理

当多个五行得分接近时 (差距 < 0.1):

1. 优先参考颜色维度
2. 考虑衣物的整体气质
3. 参考品牌定位和目标人群
