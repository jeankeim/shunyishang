# Task 01 验收标准

## 功能验收

### 1. 响应式断点测试

| 断点 | 屏幕宽度 | 预期行为 | 状态 |
|:---|:---|:---|:---:|
| 移动端小屏 | 320px - 374px | 顶部导航 + Drawer 侧边栏 | ⬜ |
| 移动端标准 | 375px - 767px | 顶部导航 + Drawer 侧边栏 | ⬜ |
| 平板端 | 768px - 1023px | 可收起侧边栏 | ⬜ |
| 桌面端 | ≥ 1024px | 固定侧边栏 (260px) | ⬜ |

### 2. 移动端导航测试

| 测试项 | 预期结果 | 状态 |
|:---|:---|:---:|

| Hamburger 按钮点击 | 打开/关闭菜单 | ⬜ |

| 点击遮罩层 | 关闭菜单 | ⬜ |
| 菜单动画 | 流畅无卡顿 | ⬜ |

### 3. 侧边栏测试

| 测试项 | 移动端 | 桌面端 | 状态 |
|:---|:---:|:---:|:---:|
| 默认显示 | Drawer (隐藏) | 固定显示 | ⬜ |

## 视觉验收

### 1. 布局检查

```bash
# Chrome DevTools 检查清单
- [ ] 无水平滚动条
- [ ] 内容不溢出视口
- [ ] 间距合理 (padding/margin)
- [ ] 字体大小可读
- [ ] 图片不变形
```

### 2. 断点切换

- [ ] 从桌面端缩小到移动端，布局平滑过渡
- [ ] 无布局抖动 (CLS < 0.1)
- [ ] 无内容闪烁

### 3. 视觉一致性

| 元素 | 移动端 | 桌面端 | 状态 |
|:---|:---|:---|:---:|
| 主内容区 | `ml-0 mt-14` | `ml-64 mt-0` | ⬜ |
| 侧边栏 | Drawer | Fixed 260px | ⬜ |
| 顶部导航 | 固定 56px | 隐藏 | ⬜ |
