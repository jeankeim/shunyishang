## 💡 使用指南

### 如何开始一个新任务？

1. **读取任务文档**:
   ```bash
   cat TASKS/WEEK_07_5_MOBILE_RESPONSIVE/README.md
   cat TASKS/WEEK_07_5_MOBILE_RESPONSIVE/01_LAYOUT_NAVIGATION/task_instruction.md
   ```

2. **按照任务指令开发**:
   - 遵循 `task_instruction.md` 中的实现步骤
   - 参考代码示例和技术要求

3. **验收测试**:
   - 按照 `acceptance_criteria.md` 逐项测试
   - 记录测试结果和问题

4. **更新状态**:
   - 标记任务为完成
   - 更新本文档
