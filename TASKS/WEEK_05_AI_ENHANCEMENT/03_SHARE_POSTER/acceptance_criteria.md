## 模板详细规格

### 模板1: 简约风格
```
尺寸: 1080 x 1920 px
背景: 纯色/渐变
字体: 思源黑体 / PingFang SC
配色: 白底黑字 / 黑底白字 / 五行色
元素:
  - 标题区 (顶部居中)
  - 主图区 (中部，占比40%)
  - 单品列表 (主图下方)
  - 五行信息 (底部)
  - 品牌标识 (右下角)
```

### 模板2: 五行风格
```
尺寸: 1080 x 1920 px
背景: 五行色渐变/纹理
字体: 思源宋体 / 传统字体
配色: 金色/红色/木质色
元素:
  - 五行图标 (顶部)
  - 主标题 (传统排版)
  - 穿搭图片 (中部)
  - 五行解析 (图文结合)
  - 二维码 (底部)
```

### 模板3: 卡片风格
```
尺寸: 1080 x 1920 px
背景: 白色/浅灰
字体: 系统默认
配色: 多彩/活泼
元素:
  - 用户头像/名称 (顶部)
  - 文案 (用户输入)
  - 单品网格 (2x2或1x4)
  - 互动数据 (底部)
  - 标签 (底部)
```
