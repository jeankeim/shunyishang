# Week 6 - Task 02: Docker Compose 生产配置 - 验收标准

## 镜像验收标准

### 1. 后端镜像
| 验收项 | 验收标准 | 测试方法 |
|--------|----------|----------|
| 镜像大小 | < 500MB | docker images |
| 构建时间 | < 5分钟 | 计时 |
| 层数 | < 15层 | docker history |
| 无构建依赖 | 无 gcc/python-dev | 检查镜像 |
| 非 root 用户 | 使用 appuser | docker inspect |

### 2. 前端镜像
| 验收项 | 验收标准 | 测试方法 |
|--------|----------|----------|
| 镜像大小 | < 100MB | docker images |
| Nginx 版本 | 使用 alpine | docker inspect |
| 静态资源 | Gzip 压缩 | curl 测试 |
| 缓存配置 | 静态资源缓存1年 | 检查配置 |

### 3. 安全扫描
| 验收项 | 验收标准 | 测试方法 |
|--------|----------|----------|
| 高危漏洞 | 0个 | Trivy/Clair扫描 |
| 中危漏洞 | < 5个 | Trivy/Clair扫描 |
| 敏感信息 | 无硬编码密钥 | 代码审查 |
| 权限设置 | 文件权限正确 | docker inspect |
