完成本周后，系统应具备以下能力：

| 标准 | 描述 |

## 💡 难点与提示

### 1. 向量权限控制
```sql
-- ❌ 错误：向量搜索不带权限
SELECT * FROM user_wardrobe ORDER BY embedding <=> $1;

-- ✅ 正确：WHERE 子句强制加 user_id
SELECT * FROM user_wardrobe 
WHERE user_id = $1 AND embedding IS NOT NULL
ORDER BY embedding <=> $2;
```

### 2. AI 识别不准确怎么办？
- 提供编辑功能，允许用户手动修正
- 修正后重新生成 Embedding 并更新

### 3. 冷启动问题
- 新用户衣橱为空时，Agent 应引导用户添加衣物
- 或自动进入"购物推荐模式"

### 4. 性能优化
- `user_id` 字段建立 B-tree 索引
- HNSW 索引用于向量搜索
- 考虑使用 `EXPLAIN ANALYZE` 验证查询计划
